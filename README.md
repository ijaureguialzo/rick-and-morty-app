# 📻 Interdimensional Radio — Rick and Morty App

A creative single-page web app themed as an **Interdimensional Radio** that tunes into different dimensions of the Rick and Morty universe.

:warning: Completely bogus 100% AI made test website. 

## Features

- **5 Dimensions** to tune into: Characters, Episodes, Locations, Quotes (random), and Favorites
- **Retro Radio UI** with CRT scanline effects, neon green glow, and portal animations
- **Real Data** from the Rick and Morty API — browse characters, episodes, and locations
- **Interdimensional Cable** — Random quote generator with static/noise animation
- **Favorites System** — Save your favorite characters (persists in localStorage)
- **Portal Transitions** — Green portal animation when switching dimensions

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will open at `http://localhost:3000`.

### Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- **Vite** — Fast build tool and dev server
